# Online-Ticket-Booking-System
The Online Ticket Booking System is a Python-based application that uses a MySQL database to manage event bookings efficiently. It allows users to register, log in, view available events, and book tickets based on seat availability. The system ensures smooth booking operations by updating seat counts in real time.

A key feature of this system is the generation of QR codes for each booking, which can be used for secure ticket verification. Users can also cancel their bookings, and the system automatically updates the available seats. Additionally, an admin view is provided to monitor all bookings and user activity.

Overall, this project demonstrates how Python and database integration can simplify the ticket booking process. It reduces manual effort, improves accuracy, and can be further enhanced with features like online payments, web interface, and advanced security.
